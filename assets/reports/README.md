# Test-report graphics

The PDF test report (js/testReport.js) embeds three 406 MHz analyser screenshots.
Save the three images here with these **exact** filenames:

| File | Image |
|---|---|
| `phase-406.jpg`    | "Phase 406 at time" — complex modulation plot |
| `power-406.jpg`    | "Power 406 at time" — burst power envelope |
| `spectrum-406.jpg` | "Spectrum 406" — carrier peak with mask |

If a file is missing, the report still generates but shows a grey placeholder box
in that spot. These images are committed to the repo so GitHub Pages can serve them.
